y = x² - The Quadratic Function & Parabola

Introduction to y = x²

The function y = x², also written as f(x) = x², is the parent quadratic function. It takes every input number x and returns its square. Because a positive number squared and a negative number squared both produce a non-negative result, the graph never goes below the x-axis.[1]

Visually, y = x² forms a smooth U-shaped curve called a parabola. Its lowest point is the vertex at (0, 0), and the y-axis is its line of symmetry. This simple function is the base model for every quadratic expression of the form y = ax² + bx + c.

Origin and Historical Context

The idea of squaring numbers is ancient. Babylonian mathematicians used tables of squares thousands of years ago to support arithmetic, measurement, and early equation solving. Geometry also gave quadratic thinking a natural home: finding the area of a square requires multiplying a side length by itself, so area = x².

The curved shape connected to x² was studied in classical Greek geometry as a parabola, especially through work on conic sections. In the 17th century, coordinate geometry connected algebra and curves, making it natural to draw equations such as y = x² on an x-y plane.

{\displaystyle f(x)=x^{2},\quad f(-x)=f(x),\quad f'(x)=2x}

Core Properties

The domain of y = x² is all real numbers, because any real input can be squared. The range is y ≥ 0, because squares cannot be negative in the real-number system. The function is even, meaning f(-x) = f(x); for example, (-4)² = 16 and 4² = 16.

The slope changes as x changes. Near the vertex the curve is flat, while farther from the origin it becomes steeper. In calculus, this changing slope is described by the derivative f'(x) = 2x.

Worked Examples

If x = 5, then y = 5² = 25. If x = -5, then y = (-5)² = 25. This pair of equal outputs shows the symmetry of the parabola.

A small table of values helps reveal the shape:

x: -3 -2 -1 0 1 2 3 y: 9, 4, 1, 0, 1, 4, 9

Scientific and Engineering Uses

Quadratic functions appear in uniformly accelerated motion. For example, the distance traveled by an object under constant acceleration often contains a squared time term, such as d = ½gt². This is why falling objects and projectiles are commonly modeled using quadratic equations.[2]

Engineers use parabolas in reflectors because a parabolic shape can focus parallel incoming rays toward one point. This principle appears in satellite dishes, radio antennas, solar cookers, telescopes, microphones, and headlights.

Real-Life Uses

Area: The simplest real-life meaning of x² is square area. If a square has side length x, its area is x². Doubling the side length does not double the area; it makes the area four times as large.

Sports and fountains: The path of a basketball shot, a thrown ball, or a water stream is often approximated by a parabola when air resistance is ignored.

Design and construction: Parabolic arches and bridge cable models use quadratic shapes because they distribute load and create stable, predictable curves.[3]

Optimization: Quadratic models help find maximum profit, minimum cost, best dimensions, and efficient designs. Since a parabola has a clear turning point, it is useful when a problem has one best maximum or minimum value.

How Transformations Work

The parent function y = x² can be moved, stretched, compressed, or flipped. In y = a(x - h)² + k, the point (h, k) becomes the vertex. If a is positive, the parabola opens upward; if a is negative, it opens downward.
